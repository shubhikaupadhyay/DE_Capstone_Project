# gx_data_docs/

The actual Great Expectations Data Docs site generated by
`notebooks/great_expectations.ipynb`, downloaded from Databricks and
checked in so the validation results can be browsed directly.

## What's here

| Path | Contains |
|---|---|
| `index.html` | Entry point — lists every validation run against the Silver orders table |
| `expectations/silver_orders_suite.html` | The expectation suite definition: every rule checked (not-null `order_id`/`customer_id`, `order_amount` > 0, `order_date` in range, referential integrity against customers) |
| `validations/silver_orders_suite/` | The actual validation run results, one page per run, showing pass/fail per expectation |
| `static/` | Fonts, images, and stylesheets the site needs to render |
